# Guide d'Installation - Simulation Épidémie

Ce projet est une simulation interactive d'épidémie développée en Python (Tkinter). Suivez ces étapes simples pour lancer la simulation sur votre ordinateur.

---

## 1. Téléchargement
Récupérez le dossier nommé **"version final"** et enregistrez-le directement sur votre ordinateur (sur votre Bureau ou dans vos Documents).

> **Note :** Ne lancez pas le script depuis un fichier `.zip`. Il faut impérativement extraire le dossier pour que les images se chargent correctement.

## 2. Prérequis
Assurez-vous d'avoir **Python 3** installé sur votre machine. 
*Aucune bibliothèque externe n'est requise (le projet utilise `tkinter`, `random`, `os` et `ctypes` qui sont inclus de base).*

## 3. Lancement
1. Ouvrez le dossier **"version final"**.
2. Localisez le fichier nommé `main.py`.
3. Lancez-le :
   - **Windows :** Double-cliquez sur `main.py`.
   - **Via Terminal/Console :** ```bash
     python main.py


## Utilisation
- **Réglages :** Utilisez les curseurs à droite pour modifier la propagation en temps réel.
- **Afficher Carte :** Active le fond de carte (assurez-vous que le dossier `images` contient bien `map1.png`).
- **Reset :** Relance une nouvelle simulation avec les paramètres actuels.
